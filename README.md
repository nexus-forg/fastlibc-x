# fastlibc-x

Высокопроизводительные расширения системных примитивов C, оптимизированные для современных процессоров.

## Почему это круто?
Цель проекта — предоставить альтернативные реализации стандартных функций (например, `memcpy`), которые работают быстрее за счет использования SIMD-инструкций и низкоуровневой оптимизации.

## Быстрый старт
1. Клонируй репозиторий: `git clone https://github.com/nexus-forge/fastlibc-x.git`
2. Собери проект: `make` (или используй предоставленные скрипты)
3. Запусти тесты: `make test`
